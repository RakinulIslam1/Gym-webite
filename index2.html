Question 1: What's the difference between var, let and const

Answer: The var is the oldest keyword to declare a variable in JavaScript.let is block-scoped. var is function scoped. let does not allow to redeclare variables. var allows to redeclare. const` is a signal that the identifier won't be reassigned and `let` is a signal that the variable may be reassigned, such as a counter in a loop, or a value swap in an algorithm. It also signals that the variable will be used only in the block it's defined in, which is not always the entire containing function. variables. var can be declared without initialization. let can be declared without initialization. const cannot be declared without initialization.

Question 2: Difference between Arrow function in regular function

Answer: Unlike regular functions, arrow functions do not have their own this . The value of this inside an arrow function remains the same throughout the lifecycle of the function and is always bound to the value of this in the closest non-arrow parent function..
